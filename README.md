# ROS_MULTI
ros多机建图+导航

涉及到ros小车的建图和导航功能

常用命令可见 command_multi

multi_robot: 上位机程序，主要包括多机建图和导航的launch文件和param文件

control：项目中的控制文件，参考意义不大

attachment：主程序运行中记录数据的文件

Iterface：我们项目中开发的用户交互界面，使用工具qt-5

clbrobot/imu_calib：修改过后的小车程序，以支持多机导航
